# Ultra Memo

**最強・最高・最速** のメモアプリケーション。

ミニマルな執筆体験を追求した、Tauri v2 + Rust + SQLite によるデスクトップアプリです。

## ✨ 特徴

- **漆黒のエディタ**: 黒背景に白文字のみ。視覚的なノイズを排除した究極の執筆環境
- **ツリー構造**: ノートを階層で管理し、思考を整理
- **FTS5 全文検索**: `Ctrl+P` で瞬時に全ノートを検索、スニペット表示付き
- **タブレス 2ペイン**: 左右にノートを開いて比較・編集
- **マークダウンプレビュー**: WYSIWYG 風のマークダウン表示モード（チェックボックス対応）
- **Wikiリンク**: `[[ノート名]]` でノート間をつなぎ、クリックでジャンプ
- **クイックキャプチャ**: 一言メモを即座に積み上げ（タイトル自動・整理不要）
- **コマンドパレット**: `Ctrl+Shift+P` で全操作にアクセス
- **Zenモード**: `F11` で集中執筆
- **遅延読み込み**: ツリーは軽量、本文は開いたときだけ取得
- **ツリー仮想化**: 大量ノートでも軽快なスクロール
- **Follow Active**: アクティブノートへの自動追従
- **同期スクロール**: 2ペイン間のスクロール同期
- **Markdown エクスポート**: ツリー構造をそのまま Markdown ファイルに出力

## 🛠️ 技術スタック

| レイヤー | 技術 |
|---------|------|
| Frontend | Next.js, React, Zustand, CodeMirror 6 |
| Backend | Tauri v2, Rust |
| Database | SQLite (WAL モード, FTS5) |
| UI | shadcn/ui, Tailwind CSS |

## 🚀 セットアップ

### 必要な環境
- Node.js 18+
- Rust (rustup)
- Windows / macOS / Linux

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run tauri dev
```

## ⌨️ ショートカット

| キー | 機能 |
|------|------|
| `Ctrl+N` | 新規ノート（同階層）— エディタ内でも有効 |
| `Ctrl+Shift+N` | 新規子ノート（本文入力へ直行）— エディタ内でも有効 |
| `Ctrl+P` | 検索 (Quick Switcher) |
| `Ctrl+Shift+F` | サイドバー検索にフォーカス |
| `Ctrl+1` / `Ctrl+2` | ペイン切り替え |
| `Ctrl+Enter` | 反対のペインで開く（2ペイン時・エディタ外） |
| `Ctrl+Enter`（エディタ内） | ソフト改行（リスト継続なし） |
| `Enter` | 選択中のノートを開く（ツリーフォーカス時はエディタへ） |
| `Ctrl+Shift+P` | コマンドパレット |
| `Ctrl+Shift+M` | クイックキャプチャ（一言メモ）— エディタ内でも有効 |
| `Ctrl+Shift+[` / `]` | すべて折りたたむ / 展開 |
| `Alt+←` / `Alt+→` | 履歴で戻る / 進む（エディタ内でも有効） |
| マウス戻る / 進む | 履歴で戻る / 進む |
| `F11` | Zenモード |
| `Ctrl+F` | ファイル内検索 |
| `Ctrl+S` | 即時保存 |
| `Ctrl+B` / `Ctrl+I` | 太字 / 斜体 |
| `Ctrl+Alt+1` / `2` / `3` | 見出し H1 / H2 / H3 |
| `Ctrl+Shift+D` / `Ctrl+Shift+K` | 行を複製 / 行を削除 |
| `Mod+クリック` on `[[wiki]]` | 反対ペインで wiki リンクを開く |
| ツリーダブルクリック | ノートを開いてエディタにフォーカス |
| サイドバー検索 | ツリー内タイトル・本文プレビューをフィルタ（`Ctrl+P` の FTS とは別） |
| `F2` | インラインリネーム |
| `Shift+ドラッグ` | 子ノート化 |
